## Astro 框架简介

Astro 是一个专注于内容的静态网站框架，具有出色的性能和开发体验。

### 为什么选择 Astro？

1. ** Islands 架构**：只发送必要的 JavaScript
2. **多框架支持**：支持 React、Vue、Svelte 等组件
3. **优秀的 SEO**：默认服务端渲染
4. **极快的速度**：自动优化资源加载

### 安装 Astro

```bash
npm create astro@latest
```

### 项目结构

```
src/
├── pages/
├── components/
├── layouts/
└── content/
```

### 总结

Astro 是构建内容驱动网站的绝佳选择。
